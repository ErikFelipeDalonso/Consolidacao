O trabalho solicitado pelo docente foi estruturado em duas etapas. Na primeira etapa, foi pedido que os alunos realizassem uma pesquisa sobre 20 tags do estilo HTML, a fim de ampliar o repertório e consolidar o entendimento que havia sido aprendido. 
Nesse sentido, cada tag precisava ser explicada, sendo subdividida em seus principais atributos: a função que desempenha dentro de uma página web e os contextos em que costuma ser utilizada.
Na segunda etapa, o objetivo era apresentar exemplos simples e claros de uso e, ao mesmo tempo, aplicar os conhecimentos que haviam sido pesquisados na atividade anterior. 
O docente deixou claro a importância de relacionar a prática com a teoria, mostrando como cada elemento contribui para a construção de páginas funcionais.
Em suma, ao mesmo tempo em que se exigiu uma pesquisa, também se incentivou a aplicação prática dos conceitos, preparando-nos para compreender a lógica por trás da construção de páginas web, pesquisando páginas e inspecionando-as para encontrar os atributos e tags.
Desse modo, eu, como aluno, entendi que a proposta era que aprendessemos a ser autodidatas, pois essa é uma habilidade muito necessária para a área de DS.
